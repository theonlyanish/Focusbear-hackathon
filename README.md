# Focusbear-hackathon

A React Native application for the Focusbear hackathon.

## Getting Started

1. Clone the repository
2. Install dependencies: `npm install`
3. Run on iOS: `npx react-native run-ios`
4. Run on Android: `npx react-native run-android`
